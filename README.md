# Snowflake Resources Guide

A curated collection of Snowflake learning resources, documentation, tools, and community links.

**Live Site:** [sfc-gh-tmeacham.github.io/sf-resources](https://sfc-gh-tmeacham.github.io/sf-resources/)

## Features

- **Searchable** — Filter resources by keyword
- **Dark/Light Mode** — Theme toggle with system preference detection
- **Zero Dependencies** — Static HTML, inline CSS, vanilla JS
- **Mobile Friendly** — Responsive sidebar navigation

## Categories

- Learning & Certification
- Documentation
- Blogs & Community
- Developer Tools
- Well-Architected Framework
- And more...
